========================================
정리 가능한 빌드 디렉토리
========================================

dist/ (684MB)
- 릴리즈 빌드 결과물
- 정리 시점: 새 릴리즈 빌드 전
- 재생성: npm run build:win

out/ (2.3MB)
- 개발 빌드 결과물
- 정리 시점: 불필요 시 (자동 재생성됨)
- 재생성: npm run dev 자동 생성

node_modules/
- 의존성 패키지들
- 정리 방법: rm -rf node_modules && npm install
- 권장하지 않음 (시간 소요)
